# Claim Boundary

Use this repository to support claims about a bounded Cost-Agent scheduler:

- Cost-Agent consumes structured sequence cards, not raw images.
- Cost-Agent ranks sequences and enforces an HRTI frame budget.
- Cost-Agent selects from a finite set of validated candidate outputs.
- Cost-Agent emits auditable decisions, traces, and cost reports.

Do not use this code to claim that Cost-Agent directly detects objects, tracks objects, edits bounding boxes, corrects identities, or reasons over visual content.
