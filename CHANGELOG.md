# Changelog

## 0.1.0 - 2026-06-15

- Initial anonymous Cost-Agent artifact.
- Provides the deterministic Cost-Agent scheduler, sequence-card builder,
  verifier, traceable executor, toy runnable example, and paper support tables.
- Adds an optional structured LLM adapter for future experiments. This adapter
  is disabled by default and is not part of the paper's frozen official results.
